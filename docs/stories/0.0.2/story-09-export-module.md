# Story 09：PCM/频谱导出模块（WAV/CSV/JSON）

## 目标
- 提供可选导出模块，将波形（PCM 抽样）与频谱数据导出为 WAV/CSV/JSON，便于离线分析与对接第三方工具。

## 测试优先（TDD）
- ✖️ [1] 导出正确性单测：给定固定 PCM/频谱输入，输出文件内容与预期匹配（WAV header/数据、CSV/JSON 字段）。
- ✖️ [2] 参数与错误单测：文件路径非法/无写权限/空数据等返回预期错误码。
- ✖️ [3] 集成测试：在 demo 播放中启用导出，验证生成文件可被 PC 工具读取（如 Python 解析）。

## 开发任务
- ✖️ [4] 设计导出 API（enable/disable、目标格式、文件路径、覆盖策略），实现 WAV/CSV/JSON 写入（原生 C/C++/平台封装）。
- ✖️ [5] 将导出模块挂接波形/频谱事件流水线，支持节流/队列控制，避免阻塞主处理线程。
- ✖️ [6] 更新文档/示例：说明导出使用方式、数据格式字段、典型解析脚本。

## 完成标准（DoD）
- ✖️ [7] 单测/集成测试通过，导出文件可被标准工具读取且数据一致。
- ✖️ [8] 文档与示例更新，说明可选模块特性与性能注意事项。
